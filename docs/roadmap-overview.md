# Roadmap Overview

## Career Target

Fresher-ready full-stack developer with practical evidence across frontend, backend, database, mobile, testing, deployment, security, and AI integration.

## Learning Style

- Concept-first
- Project-based
- Interview-oriented
- Production-aware
- Portfolio-backed

## Core Loop

Theory -> technical skill -> project evidence -> GitHub portfolio -> CV update -> interview readiness.

## Week 0 Theme

Setup the system before learning the system. Week 0 should create momentum, not overload deep technical study.
