Start date: 2026-04-27

This repository is the parent hub for my full-stack developer learning path. It connects weekly study, GitHub evidence, interview preparation, career materials, and future thesis planning.

## Purpose

- Build toward intern/fresher full-stack developer readiness.
- Keep learning evidence visible through notes, repositories, commits, READMEs, and project writeups.
- Prepare a one-page English LaTeX CV backed by real project evidence.

- Use `career/` for CV source, PDF exports, cover letters, and job tracking.
- Use `templates/` whenever a new weekly log, README, ADR, or interview answer is needed.
